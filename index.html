s3m

Store streams of data in s3 buckets

Star
  • Install

    cargo install s3m
  • How it works

    s3m –help
  • The configuration file

    ~/.s3m/config.yml
  • Upload

    s3m /path/to/file s3/bucket/file
  • List Objects

    s3m ls s3/bucket
  • Delete Objects

    s3m rm s3/bucket/object
